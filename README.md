# 🌙 Athar.AI — Islamic Heritage Explorer

**Athar.AI** is an open-source Retrieval-Augmented Generation (RAG) system that helps users explore the scientific, cultural, and artistic heritage of the Islamic world — all using **free and open models**.

## ✨ Features

- Free, fully open-source stack (Hugging Face + LangChain)
- Local vector database (ChromaDB)
- Context-grounded answers with source citations
- Expandable dataset (Wikipedia, UNESCO, museums)
- Optional Streamlit UI

## 🧠 Tech Stack

- `LangChain`
- `HuggingFace Transformers`
- `ChromaDB`
- `SentenceTransformers`
- `Streamlit`

## 🚀 Run Locally

```bash
git clone https://github.com/Laaliji/athar-ai.git
cd athar-ai
```
